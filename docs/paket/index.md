# Pilih Paket

Setiap paket dirancang untuk tingkat kebutuhan operasional yang berbeda. Pilih yang paling sesuai.

## Daftar Paket

<div class="paket-card">

### <span class="badge badge-progress"><span class="badge-dot"></span>Paket Mushola Starter</span>

Untuk mushola/surau kecil dengan pengurus terbatas dan kebutuhan administrasi sederhana.

**Isi**: SOP buka-tutup, kebersihan, kas infak sederhana, inventaris, template jadwal, checklist harian/mingguan.

[Lihat Paket →](/paket/mushola-starter/)

</div>
