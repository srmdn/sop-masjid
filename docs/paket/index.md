# Pilih Paket

Setiap paket dirancang untuk skala masjid/mushola yang berbeda. Pilih yang paling sesuai.

## Daftar Paket

### 🚧 Mushola Starter
Untuk mushola kecil, mushala kantor, mushola RW, atau masjid yang baru mulai mendokumentasikan operasional.

**Isi**: SOP buka-tutup, kebersihan, kas infak sederhana, inventaris, template jadwal, checklist harian/mingguan.

[Lihat Paket →](/paket/mushola-starter/)

---

### 🔜 Masjid Kampung Basic *(v0.2)*
Untuk masjid kampung atau masjid lingkungan yang sudah punya struktur pengurus.

**Isi**: Semua isi Mushola Starter + struktur organisasi, jobdesk, SOP komunikasi, template surat.

---

### 🔜 Keuangan Masjid Kit *(v0.3)*
Modul tambahan untuk semua jenis masjid. Fokus pada pencatatan dan pelaporan keuangan.

**Isi**: SOP keuangan lengkap, template pelaporan, dashboard spreadsheet.

---

::: info Paket selanjutnya
Paket Masjid Kampung Basic, Keuangan Masjid Kit, Ramadhan Kit, dan lainnya akan menyusul setelah v0.1 selesai.
:::
