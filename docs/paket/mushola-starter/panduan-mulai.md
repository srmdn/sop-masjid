# Panduan Mulai — Mushola Starter

Panduan langkah demi langkah untuk mulai menggunakan paket Mushola Starter.

## Sebelum Mulai

Kamu butuh:
- Akses Google Docs dan Google Sheets (akun Google)
- Printer (opsional, untuk mencetak checklist)
- Koordinasi dengan pengurus mushola lainnya

## Langkah 1: Pahami SOP

Baca dulu semua SOP sebelum membuat salinan template:

1. [SOP Buka-Tutup Mushola](./sop/buka-tutup)
2. [SOP Kebersihan Harian](./sop/kebersihan)
3. [SOP Pengelolaan Kas Infak Sederhana](./sop/kas-infak)
4. [SOP Inventaris Barang Mushola](./sop/inventaris)

## Langkah 2: Salin Template

Setelah paham alurnya, salin template yang dibutuhkan:

1. [Template Jadwal Imam/Muadzin](./template/jadwal-imam) — klik "Salin Google Sheet"
2. [Template Kas Masuk-Keluar](./template/kas) — klik "Salin Google Sheet"
3. [Template Inventaris Barang](./template/inventaris) — klik "Salin Google Sheet"

Setiap template akan terbuka di Google Sheets. Pilih **File → Buat Salinan** untuk menyimpan ke akun Google kamu.

## Langkah 3: Cetak Checklist

1. [Checklist Harian Mushola](./checklist/harian) — cetak dan tempel di papan pengumuman
2. [Checklist Mingguan Mushola](./checklist/mingguan) — cetak untuk marbot/petugas

## Langkah 4: Sesuaikan

Setiap mushola berbeda. Sesuaikan:
- Nama mushola di header template
- Jumlah petugas dan rotasi jadwal
- Kategori kas (infak Jumat, infak harian, kotak amal, dll)
- Jenis barang di inventaris
- Checklist item yang tidak relevan
