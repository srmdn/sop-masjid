# Template Kas Masuk-Keluar

Template spreadsheet untuk mencatat pemasukan dan pengeluaran kas mushola.

## Cara Pakai

1. Klik link di bawah
2. Pilih **File → Buat Salinan**
3. Simpan ke Google Drive mushola kamu
4. Catat setiap transaksi harian
