# Template Inventaris Barang

Template spreadsheet untuk mencatat dan melacak inventaris barang mushola.

## Cara Pakai

1. Klik link di bawah
2. Pilih **File → Buat Salinan**
3. Simpan ke Google Drive mushola kamu
4. Daftarkan semua barang inventaris
