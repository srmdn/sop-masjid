# Kontribusi

SOP Masjid adalah project open-source. Semua orang bisa berkontribusi.

## Cara Berkontribusi

### Laporkan Masalah
Temukan kesalahan atau SOP yang kurang jelas? [Buka issue](https://github.com/srmdn/sop-masjid/issues).

### Usulkan SOP/Template Baru
Punya ide SOP atau template yang belum ada? Buka issue dulu untuk diskusi.

### Kirim Pull Request

1. Fork repo
2. Buat branch: `git checkout -b usul/namasop`
3. Commit: `git commit -m 'Tambah: SOP ...'`
4. Push dan buka Pull Request

## Panduan Konten

### SOP
- Bahasa Indonesia sederhana dan praktis
- Struktur: **Tujuan → Peralatan → Langkah → Catatan**
- Fokus operasional, bukan fiqh

### Template
- Simpan di Google Sheets/Docs project
- File Markdown di repo hanya berisi deskripsi dan link "Salin"

### Checklist
- Format: daftar centang (`- [ ]`) di Markdown
- Kelompokkan per area/waktu

## Yang Tidak Dibahas

- Perbedaan mazhab dan khilafiyah
- Isu politik
- Fatwa atau keputusan hukum Islam

Project ini fokus pada **administrasi dan operasional** masjid/mushola.
