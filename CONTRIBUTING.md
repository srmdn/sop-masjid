# Panduan Kontribusi

Terima kasih ingin berkontribusi ke SOP Masjid.

## Cara Berkontribusi

### 1. Laporkan Masalah
Temukan kesalahan, typo, atau SOP yang kurang jelas? Buka [GitHub Issue](https://github.com/srmdn/sop-masjid/issues).

### 2. Usulkan SOP atau Template Baru
Punya ide SOP atau template yang belum ada? Buka issue dulu untuk diskusi sebelum membuat pull request.

### 3. Kirim Pull Request

1. Fork repo ini
2. Buat branch: `git checkout -b usul/namasop`
3. Commit perubahan: `git commit -m 'Tambah: SOP Kebersihan Mingguan'`
4. Push: `git push origin usul/namasop`
5. Buka Pull Request

## Panduan Konten

### SOP
- Gunakan bahasa Indonesia yang sederhana dan praktis
- Hindari istilah teknis yang tidak diperlukan
- Struktur SOP: Tujuan → Peralatan → Langkah → Catatan
- Fokus pada aspek operasional, bukan fiqh atau khilafiyah

### Template
- Template spreadsheet: simpan di Google Sheets project, berikan link "Salin" di file Markdown
- Template dokumen: simpan di Google Docs project, berikan link "Salin" di file Markdown
- Jangan upload file `.xlsx`, `.docx`, atau PDF langsung ke repo

### Checklist
- Format: daftar centang (`- [ ]`) dalam Markdown
- Kelompokkan per area/waktu yang logis

## Kode Etik

- Tidak membahas khilafiyah, perbedaan mazhab, atau isu politik
- Fokus pada administrasi dan operasional masjid/mushola
- Hormati kontributor lain
